# Medical RAG Chatbot

This project is a Retrieval-Augmented Generation (RAG) chatbot built using:

- Streamlit
- LangChain
- FAISS
- Ollama (llama3)

## How to Run

1. Install Ollama
2. Run: ollama pull llama3
3. Run: streamlit run app.py

The chatbot retrieves answers from the provided medical PDF.
